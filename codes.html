Code & Tools

This page highlights selected scripts, analysis tools, and open-source resources I’ve developed for climate science, atmospheric dynamics, and model diagnostics.

Regional Hadley Circulation Diagnostics

Python tools for computing zonal-mean and basin-specific Hadley circulation strength (HCS) using divergent meridional wind, following Li et al. (2023).

  • Supports CESM, ERA5, MERRA-2
  • Computes ψ (streamfunction) for global and regional overturning
  • Includes trend analysis and visualization utilities

View on GitHub

ENSO–Atlantic Variability Analysis

Scripts for evaluating ENSO influence on Atlantic SST variability using CESM2 Fully-Coupled and NoENSO simulations.

  • EOF and PC analysis with xeofs
  • Variance ratio maps
  • Lagged correlations and composites

View on GitHub

Monsoon Dynamics: ENSO, IOD & Atlantic Niño

A collection of analysis notebooks for diagnosing SASM variability and disentangling contributions from ENSO, the Indian Ocean Dipole, and Atlantic Niño.

  • Seasonality and power spectrum diagnostics
  • Cross-basin influence detection
  • Process-based interpretation
